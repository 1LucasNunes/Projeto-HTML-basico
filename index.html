Estrutura básica do HTML

< voltar

Basicamente as paginas HTML são compostas por tags e dividos em 3 partes pricipais conhecidas como, html, head e body.

Html

Podemos dizer que a tag html é o tipo de conteúdo do documento da respectiva linguagem

Head

A tag head será o cabeçalho, onde são armazenadas as inforções que não aparecerão ao usuário. É nessa tag que colocamos as tags como, title (titulo que apare na aba ou janela), style (para alterar os estilos na pagina), scripts que são as alterações que faremos em JavaScript, que veremos mais tarde, entre outras tags.

Body

Body é como o próprio nome já diz, o corpo da pagina, onde vai ser colocado tudo que aparecerá ao usuario, tudo que ele pode ver, imagens, videos, textos, etc.

Tags populares

< voltar

Agora vamos ver algumas tags muito usadas em estruturas html. Lembrando que todas as tags são descritas por sinal de menor (<), o nome da tag e um sinal de maior (>). Algumas tags necessitam de um inicio e um fechamento, iniciando como descrito anteriormente e fechando com uma barra (/) depois do sinal de menor e antes do nome da tag.

  • p - permite escrever um paragrafo
  • a - ancora (leva à algum lugar)
  • input - cria diferentes campos de entrada de dados de acordo com atributos
  • h - define titulos e subtitulos
  • img - insere uma imagem

Tags para texto

< voltar

Pensando nos textos, existem algumas tags especificas que alteram apenas as palavras, ou inserem um comportamento para aquela palavra ou texto. A tag blackquote por exemplo, que da um formato diferente a frase, quando se quer fazer uma citação ou uma frase de impacto, a própria tag h que em conjunto com um numero, define tamanhos diferentes para titulos e subtitulos.

Atributos de Tags

< voltar

Algumas tags tem atributos que são usados para acrescentar determinada informação a elas, ou seja, controlar o comportamento de um elemento. Temos alguns exemplos de atributos bastante usados como o atributo type da tag input, tambem o atributo src da tag img, e muitos outros.

Lista Ordenada e Lista não ordenada

< voltar

Essas são listas muito utilizadas no html, uma (lista ordenada) traz uma representação sequencial numérica dos itens que se encontram nela, a outra (lista não ordenada) faz uma marcação em cada item. Serve muito bem para fazer um índice, ou descrever passos de uma receita, e seus próprios ingredientes, e mais uma infinidade de coisas.

Falando sobre links

< voltar

Como sabemos, hoje navegando na internet, vemos os links por toda parte, mas como é que eles funcionam dentro do html?. Sabemos que ao clilcar em determinada parte de uma pagina na web, somos enviados a outras paginas, mas como isso é feito, é que deixa tudo mais interessante. Usamos a tag a para ancorar nosso link, logo em seguida colocamos o atributo href="" e definimos entre as aspas, o documento ou a url para onde o usuario vai ser enviado. É possivel até mesmo fazer o caminho de volta, apontando para onde o usuario voltará com a mesma tag, mesmo atributo, definindo apenas o novo documento ou url.
